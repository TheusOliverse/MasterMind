  • MasterMind - Um ótimo lugar para aprender.

    Não é muito comum ver uma plataforma que misture poesia, arte, diversão, ciência e tecnologia. Dedico este projeto para aqueles que gostam de estudar e que sabem que todo o esforço não é uma perda de tempo. Existe uma pequena porcentagem de estudantes que não consegue se destacar, já que estes não estão onde deveriam estar. Não é todo mundo que percebe o potencial de quem realmente se esforça, e assim fiz um grupo para juntar os estudantes.
